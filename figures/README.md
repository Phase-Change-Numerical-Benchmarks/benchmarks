# Reference Figures

Each benchmark stores its reference plot as `figures/<ID>-reference.svg`.

All figures are generated by `scripts/plot_reference_figures.py`; do not
edit the SVG output by hand. To regenerate a single case:

```bash
python3 scripts/plot_reference_figures.py PA-010
```

or omit the argument to regenerate everything. Problem sketches (non-plot
schematics) may be added as separately named SVG files and referenced from
the case front matter.
